Gpt4支持的文件上传类型

Posted on 2024-06-10 Edited on 2024-11-16 In AI , 科技

ChatGPT-4 supports uploading a variety of file types for processing and analysis. Here are some common file types you can upload:

  1. Text Files: .txt, .csv
  2. Documents: .pdf, .docx, .xlsx, .pptx
  3. Images: .jpg, .jpeg, .png, .gif
  4. Code Files: .py, .java, .cpp, .js, .html, .css, etc.

If you have a specific type of file you want to upload, let me know, and I can confirm if it is supported.
