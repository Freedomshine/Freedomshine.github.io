Markdown语法

Created2024-06-10|Updated2024-11-16|科技
|Post Views:

Markdown 是一种轻量级标记语言，常用于编写格式化的文本，特别适合用于笔记、文档、博客和论坛等场景。以下是一些常用的 Markdown 语法：

标题

Markdown 使用 # 符号表示标题。一个 # 表示一级标题，两个 # 表示二级标题，依此类推。

段落

段落之间用一个空行分隔。

加粗和斜体

使用 * 或 _ 表示斜体，使用 ** 或 __ 表示加粗。

列表

无序列表

使用 -、+ 或 * 表示无序列表项。

有序列表

使用数字后跟一个点表示有序列表项。

链接

使用方括号 [ ] 包裹链接文本，圆括号 ( ) 包裹链接地址。

图片

语法与链接相似，只是在前面加一个感叹号 !。

引用

使用 > 表示引用。

代码

行内代码

使用反引号 ` 包裹代码。

代码块

使用三个反引号 ````` 包裹多行代码，可以指定代码语言以启用语法高亮。

这是一个代码块。

分割线

使用三个或更多的 -、* 或 _ 创建分割线。

注释

Markdown 本身不支持注释，但可以使用 HTML 注释语法。

这些是 Markdown 语法的基础部分，了解这些可以帮助你在大多数情况下编写格式化的文本。Markdown 还支持一些扩展语法，如任务列表、表情符号等，具体取决于所使用的平台和渲染器。
