Supervised Learning和Unsupervised

Created2024-10-22|Updated2024-11-16|技术分享
|Post Views:

Supervised Learning: Regression 和 Unsupervised Learning 的主要区别在于数据是否有标签以及任务目标不同。

1. Supervised Learning: Regression（监督学习：回归）

监督学习中的回归是指算法从带有标签的数据中学习，目标是预测连续的数值输出。例如，给定一些输入数据，系统需要预测一个连续的值，如房价、温度、收入等。

  • 数据特点：在回归问题中，训练数据集包含输入特征（features）和对应的已知目标值（标签）。

    • 例如，房价预测问题中，输入特征可能是房子的面积、房间数量、位置等，目标值是房子的价格。

  • 任务目标：回归任务的目标是建立一个模型，能够根据新的输入数据预测其对应的连续数值。

    • 举例：预测未来几天的股票价格或某个客户在未来的消费金额。

  • 常用算法：

    • 线性回归（Linear Regression）
    • 多项式回归（Polynomial Regression）
    • 支持向量回归（Support Vector Regression, SVR）

  • 示例：

    • 预测汽车的油耗（数值型输出）
    • 根据过去的天气数据预测未来的温度

  • 输入与输出：

    • 输入：多维特征（例如，年龄、身高等）
    • 输出：连续的数值（例如，价格、时间等）

2. Unsupervised Learning（无监督学习）

无监督学习的特点是使用无标签的数据，目标是发现数据中的隐藏模式或结构。没有明确的输出目标值，算法需要自己寻找数据的模式或群组。

  • 数据特点：无监督学习的数据没有标签，也就是说数据集中只有输入特征，没有与之对应的目标值。算法需要自己找到数据的结构、模式或关系。

    • 例如，在客户数据中，没有给出客户是否是高消费客户，算法需要自动找出不同类型的客户。

  • 任务目标：无监督学习通常用于发现数据中的群组、关联或异常点。它主要用于聚类、降维或异常检测任务。

    • 举例：在客户数据中发现相似客户的群组，或者在银行交易数据中检测异常的交易行为。

  • 常用算法：

    • 聚类算法（Clustering）：如 K-Means、层次聚类（Hierarchical Clustering）
    • 主成分分析（Principal Component Analysis, PCA）：用于降维
    • 密度估计（Density Estimation）
    • 异常检测（Outlier Detection）

  • 示例：

    • 对客户进行聚类分析，找出具有相似购买行为的客户群体。
    • 在网络流量数据中检测异常流量（如潜在的网络攻击）。

  • 输入与输出：

    • 输入：多维特征（例如，客户的年龄、消费习惯等）
    • 输出：数据的模式、群组或异常点（例如，将客户分为不同的群组）

关键区别：

特点

Supervised Learning: Regression

Unsupervised Learning

数据标签

有标签数据，输入数据与目标值一一对应

无标签数据，只有输入特征，没有目标值

任务目标

预测连续的数值，例如价格、时间等

发现数据中的隐藏模式、群组或异常点

常见任务

预测问题，如房价预测、温度预测

聚类、降维、异常检测等

常用算法

线性回归、多项式回归、支持向量回归等

K-Means、层次聚类、主成分分析等

输出类型

连续数值

数据结构（群组、降维后的特征）

总结：

  • 监督学习：回归 是通过学习有标签的输入输出数据来预测连续的数值。
  • 无监督学习 是在没有目标值的情况下，通过算法来识别数据中的模式和结构，例如聚类或异常检测。
