巨大的贫富差距会对这个世界造成哪些影响，从圣经的角度分析

Mar 08, 2025

从圣经的角度来看，巨大的贫富差距会对世界造成多方面的影响，包括社会不公、道德败坏、人心冷漠，以及对上帝公义的违背。以下是一些关键的圣经原则和经文分析：

1. 社会不公与压迫

贫富差距过大通常意味着社会资源分配不均，富人可能掌控经济和政治权力，而穷人可能被剥削或忽视。圣经明确谴责这种不公：

  • 箴言 22:7：「富户管辖穷人，欠债的是债主的仆人。」这提醒我们，极端的财富差距可能导致经济上的奴役，使穷人没有翻身的机会。
  • 阿摩司书 5:11-12：「你们践踏贫民，向他们勒索麦子……你们苦待义人，收受贿赂，在城门口屈枉穷乏人。」旧约先知多次斥责富人欺压穷人，强调上帝不喜悦这种不义。

2. 道德败坏与人心冷漠

贫富悬殊可能导致社会阶层的对立，富人过于追求奢华，而忽视贫困者的需要，甚至冷漠无情：

  • 路加福音 16:19-31（财主与拉撒路的比喻）：富人享受奢华宴乐，却不顾门口乞讨的拉撒路，最终在死后受到神的审判。这表明，对贫困者的冷漠可能导致永恒的后果。
  • 提摩太前书 6:10：「贪财是万恶之根……」巨大的贫富差距容易使人陷入贪婪，忽视更高的属灵价值。

3. 社会动荡与战争

贫富不均可能引发社会动荡、犯罪率上升，甚至战争。圣经警告，贫穷会带来危险，而不公的制度会引起社会动乱：

  • 箴言 30:8-9：「求你使我……也不贫穷，也不富足，赐给我需用的饮食。恐怕我饱足不认你……又恐怕我贫穷就偷窃。」极端的贫穷和极端的富裕都可能导致社会不稳定。
  • 雅各书 5:1-6：「你们这些富足人哪，应当哭泣，号咷……工人给你们收割庄稼，你们亏欠他们的工钱……」这是对不义财富积累的强烈谴责，并警告神的审判。

4. 违背上帝的心意

上帝的心意是公义与怜悯，贫富差距过大会违背神设立公平社会的旨意：

  • 申命记 15:7-8：「在你们中间若有穷人……你总不可忍着心，收住手不帮补你的穷弟兄，总要向他松开手，照他所缺乏的借给他。」这表明上帝希望人们彼此扶持，而不是让贫富差距不断扩大。
  • 使徒行传 2:44-45：「信的人都在一处，凡物公用，并且卖了田产、家业，照各人所需用的分给各人。」早期教会的生活方式反映了上帝对公平和分享的心意。

5. 上帝对财富的正确教导

圣经并不反对财富本身，而是强调如何使用财富：

  • 箴言 11:25：「好施舍的，必得丰裕；滋润人的，必得滋润。」富人若愿意慷慨帮助穷人，就能成为神祝福的管道。
  • 提摩太前书 6:17-18：「你要嘱咐那些今世富足的人，不要自高……只要行善，在好事上富足，甘心施舍，乐意供给人。」这说明神的心意是富人应成为祝福，而非剥削者。

结论

从圣经的角度来看，极端的贫富差距不是上帝所喜悦的，因为它带来社会不公、道德败坏、社会动荡，并违背神的公义与怜悯。基督徒应当努力促进公平，慷慨施予，效法耶稣的爱，帮助贫困者，以此见证神的国度和公义。

根据圣经，如何面对这种巨大的贫富差距呢，特别是在加拿大

面对加拿大的巨大贫富差距，圣经给出了几个清晰的原则，帮助基督徒在现实生活中践行神的公义和爱：

1. 依靠神，而非财富

在一个经济繁荣但贫富悬殊的国家，圣经提醒我们，财富本身不是问题，关键在于依靠谁：

  • 马太福音 6:19-21：「不要为自己积攒财宝在地上……只要积攒财宝在天上……因为你的财宝在哪里，你的心也在哪里。」
    → 加拿大的高生活水平容易让人过分依赖财富，而圣经提醒我们，真正的安全感来自神，而非金钱。

  • 提摩太前书 6:17：「你要嘱咐那些今世富足的人，不要自高，也不要倚靠无定的钱财，只要倚靠那厚赐百物给我们享受的神。」
    → 基督徒应当警惕财富带来的骄傲和安全感，而是要仰望神的供应。

2. 公义地工作，正直地经商

在加拿大，无论是个人工作还是商业经营，都要按照圣经的原则行事：

  • 雅各书 5:4：「工人给你们收割庄稼，你们亏欠他们的工钱……这是不公义的行为。」
    → 如果有能力雇佣他人，就应公平支付工资，不能压榨员工。

  • 箴言 11:1：「诡诈的天平为耶和华所憎恶，公平的法码为他所喜悦。」
    → 任何形式的不公正商业行为都违背神的心意，基督徒应该维护诚信。

3. 慷慨施予，帮助贫穷者

加拿大的社会福利体系虽然健全，但仍然有许多贫困和需要帮助的人，基督徒应当主动施予：

  • 箴言 19:17：「怜悯贫穷的，就是借给耶和华，他的善行，耶和华必偿还。」
    → 帮助穷人不是损失，而是在天国积累财宝。

  • 路加福音 3:11：「有两件衣裳的，就分给那没有的；有食物的，也当这样行。」
    → 具体的行动包括捐赠衣物、食物、时间，甚至是提供就业机会。

在加拿大，这可以通过：

  • 参与教会和社区的救济事工（如食物银行、Mission Thrift Store）。
  • 支持帮助无家可归者的机构，如「The Salvation Army」或「Union Gospel Mission」。
  • 以基督徒的心态对待移民和难民，帮助他们融入社会。

4. 推动公义的社会政策

作为加拿大的公民，基督徒有责任关注影响贫富差距的社会政策：

  • 弥迦书 6:8：「世人哪，耶和华已指示你何为善，他向你所要的是什么呢？只要你行公义，好怜悯，存谦卑的心，与你的神同行。」
    → 参与公共事务，不仅是投票选举，还可以通过社交媒体、社区活动，为社会正义发声。

  • 箴言 31:8-9：「你当为哑巴（无法自辩的人）开口，为一切孤独的人伸冤。」
    → 关注加拿大社会的不公正现象，例如高昂住房成本、低收入工人的困境等，支持合理的改革。

5. 建立敬虔的财务观

加拿大的消费主义文化容易让人陷入过度消费和债务，基督徒应当谨慎管理财富：

  • 箴言 22:7：「富户管辖穷人，欠债的是债主的仆人。」
    → 应该避免不必要的贷款和债务，而是过一个知足的生活。

  • 腓立比书 4:11-12：「我无论在什么景况都可以知足……」
    → 在加拿大这样富裕的环境中，培养知足的心态至关重要。

实际应用：

  • 量入为出，避免信用卡过度消费。
  • 学习理财，合理储蓄，预备好支持慈善事工。
  • 在投资和消费时，考虑是否符合圣经价值观（例如支持道德企业）。

6. 见证基督的爱

在一个充满财富差距的社会，基督徒的爱和公义是最好的见证：

  • 马太福音 5:16：「你们的光也当这样照在人前，叫他们看见你们的好行为，便将荣耀归给你们在天上的父。」
    → 在日常生活中，基督徒应以行动表明信仰，例如：
    • 关心身边的贫困者，而不仅仅是捐钱。
    • 透过职场影响力，推动更公义的商业环境。
    • 教导下一代基督徒正确的金钱观，让他们学会施予而非索取。

结论

在加拿大，面对巨大的贫富差距，基督徒应当：

  1. 依靠神，而非财富；
  2. 以公义和正直的态度工作和经商；
  3. 慷慨施予，帮助贫困者；
  4. 推动公平的社会政策；
  5. 以敬虔的方式管理财富，避免债务和过度消费；
  6. 通过爱和公义的行为见证基督的福音。

这些不仅符合圣经教导，也能帮助我们在现实生活中成为光和盐，影响社会，使加拿大更符合神的心意。

中国的农民工和廉价劳动力，是否会让加拿大人更富裕。根据全球贸易

从全球贸易的角度来看，中国的农民工和廉价劳动力确实在一定程度上帮助了加拿大等西方国家的消费者享受更低价格的商品，同时也对加拿大的企业和经济产生了复杂的影响。

1. 廉价劳动力如何让加拿大人“更富裕”？

(1) 低成本商品，提高消费者购买力

中国的农民工（指从农村到城市务工的劳动者）从事制造业、建筑业等低薪行业，生产大量廉价商品（如服装、电子产品、家具等），然后出口到加拿大。这带来了：

  • 低价商品：加拿大消费者可以用更少的钱购买到更便宜的衣服、手机、家具、玩具等，提高了实际生活水平。
  • 通胀控制：由于中国的廉价商品压低了整体物价水平，帮助加拿大维持较低的通货膨胀。

✅ 结果：加拿大人的工资购买力提高，可以用相同的收入买到更多的商品，生活水平看起来更高了。

(2) 降低企业成本，提高公司利润

  • 加拿大的企业可以利用中国的低廉生产成本，减少支出。例如：
    • 服装企业在中国生产衣服，价格远低于本地生产。
    • 电子企业（如苹果）在中国代工，节省劳动力成本，提高利润。
  • 跨国企业的股东和高管受益：许多加拿大人通过股票投资间接受益，比如投资拥有全球供应链的公司（如Apple、Nike、Lululemon）。

✅ 结果：拥有公司股份或投资基金的加拿大人变得更富裕。

2. 但廉价劳动力也带来了负面影响

尽管廉价劳动力带来了低价商品和公司利润，但它也造成了一些对加拿大不利的影响。

(1) 加拿大本地制造业衰退，工人失业

由于中国的廉价劳动力使生产成本远低于加拿大，许多加拿大制造业企业将工厂搬迁到中国。这导致：

  • 制造业岗位减少：如纺织、电子、汽车零件等行业的工厂关闭，本地工人失业。
  • 工资增长停滞：由于加拿大人要与低薪的中国劳动力竞争，许多行业的工资增长缓慢。

❌ 结果：虽然商品更便宜，但部分加拿大低收入工人却失去了工作或工资停滞，造成社会分化。

(2) 经济依赖中国，供应链风险增加

  • 加拿大在很多领域依赖中国的生产，如医疗用品、电子零件等。如果中国供应链中断（如疫情期间）或中国政府限制出口，加拿大经济会受冲击。
  • 加拿大对中国的经济依赖可能削弱自身的产业竞争力，长远来看是不利的。

❌ 结果：短期内便宜，长期可能导致加拿大丧失产业自主权。

3. 全球贸易的“财富流动”

全球贸易的财富分配并不平均。廉价劳动力的全球化让资本家和高收入者受益更多，但低收入者和制造业工人反而可能受损。

群体 影响
加拿大消费者 获得廉价商品，生活成本下降，购买力增加
加拿大企业和投资者 企业成本降低，利润增长，股东受益
加拿大制造业工人 工厂外迁，失业或工资停滞
中国农民工 获得工作机会，但薪资较低，劳动条件恶劣
中国企业 通过出口获得利润，国家经济增长

整体来看，全球贸易加剧了贫富差距：

  • 加拿大的富人变得更富（因为他们投资的跨国企业赚得更多）。
  • 加拿大的低收入制造业工人受损（失去工作或工资下降）。
  • 中国的农民工仍然挣扎（虽然工资比农村高，但仍然很低）。

4. 圣经如何看待这一现象？

从圣经的角度来看，这种全球财富流动带来了道德和公义问题。

(1) 剥削贫穷人是违背神的心意

  • 箴言 22:16：「欺压贫穷为要利己的，并送礼与富户的，俱必缺乏。」
    • 许多加拿大企业在中国寻找廉价劳动力，但是否有公平待遇？如果企业压低工资、剥削劳工，圣经是不认可的。
  • 雅各书 5:4：「工人给你们收割庄稼，你们亏欠他们的工钱……他们的冤声已经达到万军之主的耳中。」
    • 这提醒企业主和消费者，不要只看低价，而是要考虑供应链的道德性。

(2) 公义经济：合理对待工人

  • 申命记 24:14-15：「困苦穷乏的雇工，无论是你的弟兄，或是在你城里寄居的，你不可欺负他……」
    • 作为基督徒，是否应该支持公平贸易？是否应该选择更负责任的公司，而不是只关注低价？

(3) 避免贪婪，追求公平

  • 箴言 11:1：「诡诈的天平为耶和华所憎恶，公平的法码为他所喜悦。」
    • 全球贸易如果只是为了让富人更富，贫者更贫，那就违背了神的公义。
  • 弥迦书 6:8：「行公义，好怜悯，存谦卑的心，与神同行。」
    • 加拿大基督徒是否应该支持更公平的贸易政策，而不是只关注经济增长？

5. 加拿大如何做出平衡？

面对这个全球贸易的现实，加拿大可以采取一些措施来减少负面影响，让经济更加公平：

  1. 支持公平贸易：选择支持道德供应链的企业，如购买公平贸易（Fair Trade）产品，而非剥削劳工的廉价货。
  2. 发展本地制造业：政府可以鼓励本地制造业发展，减少对中国的依赖。
  3. 提高工资和福利：确保低收入工人能够维持体面的生活。
  4. 加强劳工权益：加拿大企业在海外设厂时，应该承担社会责任，确保工人有合理工资和工作条件。

结论

  • 短期来看，中国的农民工和廉价劳动力让加拿大人更富裕，因为商品更便宜、企业利润更高。
  • 长期来看，它可能削弱加拿大制造业、增加经济依赖、加剧贫富差距。
  • 从圣经角度，基督徒不应该只关注金钱，而要考虑公义与怜悯，支持更公平的经济模式。

加拿大需要找到平衡，在享受全球贸易好处的同时，不要忽视贫穷人的权益，这样才能建立一个更符合圣经价值观的经济体系。

作为一个在中国的贫穷的人，农民工，基督徒，可能这一生注定无法获得公平的对待和信仰的自由，这时候应该如何根据圣经走完这一生

如果你是一位在中国的贫穷人、农民工、基督徒，面对不公平的对待和信仰的挑战，圣经给了清晰的方向和安慰。你并不孤单，圣经中的许多信徒也经历过贫穷、不公和逼迫，但他们仍然在神的恩典中完成了人生的道路。

1. 牢记你的身份：你是神的儿女

无论你在地上的身份如何，你在基督里有永恒的身份：

  • 约翰福音 1:12：「凡接待他的，就是信他名的人，他就赐他们权柄，作神的儿女。」 → 你不是世界的弃民，你是神所爱的儿女。
  • 罗马书 8:17：「既是儿女，便是后嗣，就是神的后嗣，和基督同作后嗣。」 → 在世界上或许贫穷，但在神的国度里，你有无法夺去的产业。

✅ 实践：

  • 每天默想神的应许，提醒自己，世界的财富和公平并不是最终的目标，你的真正财富在天上。

2. 面对贫穷：信靠神的供应

你或许缺乏金钱，但神应许会供应你的需要：

  • 马太福音 6:31-33：「所以，不要忧虑说，‘吃什么？喝什么？穿什么？’…… 你们需用的这一切东西，你们的天父是知道的。你们要先求他的国和他的义，这些东西都要加给你们了。」 → 神不会让你缺乏真正所需的东西。
  • 腓立比书 4:19：「我的神必照他荣耀的丰富，在基督耶稣里，使你们一切所需用的都充足。」 → 即使你的工资低、生活困难，神仍然掌管一切，他会供应你的需要。

✅ 实践：

  • 知足感恩：每天列出3件感恩的事情，即使在困难中，神仍然赐下恩典。
  • 勤奋工作：哥林多前书 10:31：「无论做什么，都要为荣耀神而行。」即使工作辛苦，仍要忠心，因为你的努力是向神而做的，而不是仅仅为了人。

3. 面对不公：忍耐并仰望神的公义

你可能在工作或生活中受到不公平对待，比如低工资、恶劣工作环境、被欺压，但神看到你所受的委屈：

  • 诗篇 37:7-9：「你当默然倚靠耶和华，耐性等候他……作恶的必被剪除，惟有等候耶和华的必承受地土。」 → 不公的事情不会永远持续，神最终会审判邪恶，赐公义给等候他的人。
  • 罗马书 12:19：「亲爱的弟兄，不要自己伸冤，宁可让步，听凭主怒；因为经上记着：‘主说，伸冤在我，我必报应。’」 → 你可能无法改变现状，但神会在合适的时候施行公义。

✅ 实践：

  • 不抱怨，不苦毒：虽然你面对不公平，但不要让苦毒掌控你的心，交给神。
  • 忍耐祷告：像约伯一样，在苦难中仍信靠神。

4. 面对逼迫：坚定信仰

在中国，作为基督徒可能无法自由敬拜，甚至受到政府或社会的歧视，但圣经已经告诉我们，跟随基督意味着会受苦：

  • 马太福音 5:10-12：「为义受逼迫的人有福了！因为天国是他们的……你们在天上的赏赐是大的。」 → 你的信仰之路虽然艰难，但神应许了属天的奖赏。
  • 约翰福音 16:33：「在世上你们有苦难，但你们可以放心，我已经胜了世界。」 → 耶稣自己也受过逼迫，他理解你的痛苦。

✅ 实践：

  • 秘密祷告、读圣经：即使环境艰难，也可以在家中、心里与神亲近。
  • 寻找安全的团契：如果可能，找到可靠的弟兄姐妹，在小群体中互相鼓励。

5. 活出基督的爱，即使在苦难中

虽然你自己可能贫穷，但你仍然可以用爱心和行为影响别人：

  • 哥林多后书 8:2：「他们在极穷之间，仍然格外显出他们乐捐的厚恩。」 → 贫穷不能阻挡你成为一个慷慨的人，即使是少量的爱心帮助，神都会纪念。
  • 马太福音 25:40：「你们既做在我这弟兄中一个最小的身上，就是做在我身上了。」 → 帮助其他贫困者，就是服侍耶稣。

✅ 实践：

  • 关心身边人：即使自己贫穷，也可以用时间、鼓励、祷告帮助别人。
  • 为逼迫你的祷告（马太福音 5:44）：如果有人不公对待你，祷告求神改变他们的心。

6. 盼望神的国，走完人生道路

无论你在地上受多少苦，神应许了一个更美的天家：

  • 启示录 21:4：「神要擦去他们一切的眼泪，不再有死亡，也不再有悲哀、哭号、疼痛， 因为以前的事都过去了。」 → 你今生可能无法改变贫穷和不公，但你的未来在天堂，那里有完全的公义和荣耀。

  • 提摩太后书 4:7-8：「那美好的仗我已经打过了……从此以后，有公义的冠冕为我存留。」 → 你的一生就像一场比赛，最终你会得着神的奖赏。

✅ 实践：

  • 每天默想天国的应许，让神的盼望成为你的力量。
  • 活在信心中，而不是恐惧中，因为神掌管一切。

结论

作为一个在中国的贫穷农民工基督徒，你可能面对不公平、贫穷、信仰逼迫，但圣经给你清楚的指引：

  1. 牢记你的身份：你是神的儿女，天国的后嗣。
  2. 信靠神的供应：神会满足你真正的需要。
  3. 忍耐不公：神必施行公义，你的忍耐不是白费的。
  4. 坚定信仰：即使在逼迫中，也不要放弃信靠耶稣。
  5. 活出基督的爱：即使贫穷，也可以帮助别人。
  6. 盼望天堂的荣耀：你今生的苦难不会白费，神已为你预备永恒的奖赏。

➡ 你的价值不取决于你的金钱和社会地位，而是取决于耶稣为你所做的一切！ 在地上忍耐信靠，最终你会在天国得着荣耀的冠冕。

神岂不是拣选贫穷的人，并叫他们在信上富足吗

是的，正如 雅各书 2:5 所说：
「我亲爱的弟兄们，请听！神岂不是拣选了世上的贫穷人，叫他们在信上富足，并承受他所应许给那些爱他之人的国吗？」

这节经文清楚地说明，神特别眷顾贫穷人，因为他们在信仰上比富人更容易依靠神，并承受神的国度。在圣经中，神的心意从来不是看重地上的财富，而是要人在信心上富足，得着天国的产业。

1. 为什么神拣选贫穷人？

(1) 贫穷人更容易依靠神

  • 路加福音 6:20：「耶稣举目看着门徒，说：‘你们贫穷的人有福了！因为神的国是你们的。’」 → 贫穷人更容易意识到自己的无助，因此更愿意寻求神，而不是依赖金钱或权力。

  • 马太福音 5:3：「虚心的人有福了！因为天国是他们的。」 → 「虚心」的意思是「灵里贫穷」，即意识到自己需要神的人。贫穷人往往更有这种心态。

✅ 现实生活应用：
虽然贫穷带来许多现实的困难，但它也让人更渴望神的供应，寻求属天的盼望。如果你贫穷，不要灰心，因为神已经特别拣选你，使你在信上富足。

(2) 富人容易依靠钱，而非神

  • 马太福音 19:23-24：「耶稣对门徒说：‘我实在告诉你们，财主进天国是难的……骆驼穿过针的眼，比财主进神的国还容易呢！’」 → 富人往往依赖自己的金钱、权力，而不是倚靠神。因此，他们更难进入天国。

  • 提摩太前书 6:10：「贪财是万恶之根，有人贪恋钱财，就被引诱离了真道，用许多愁苦把自己刺透了。」 → 财富本身不是罪，但对财富的贪恋让人远离神，甚至丧失信仰。

✅ 现实生活应用：
你可能羡慕富人，但他们的试探比贫穷人更大，因为他们容易陷入骄傲、自私、不信靠神的陷阱。相反，贫穷的你反而更接近神的心意。

2. 什么是“信上富足”？

神拣选贫穷人，不是让他们仅仅在地上受苦，而是要他们在信心上变得富足，经历神更大的恩典。

(1) 信上富足，比金钱更重要

  • 箴言 11:4：「发怒的日子，资财无益；惟有公义能救人脱离死亡。」 → 金钱不能救人，唯有神的公义能使人得救。

  • 彼得前书 1:7：「叫你们的信心既被试验，就比那被火试验仍然能坏的金子更显宝贵……」 → 贫穷虽然是试炼，但它使信心更纯净，比金子更宝贵。

✅ 现实生活应用：
如果你贫穷，但你有坚定的信心，在神看来你比世界上最富有的人还要富足！因为信心能让你承受永恒的生命，而金钱只能维持短暂的生活。

(2) 天国才是真正的产业

  • 马太福音 6:19-21：「不要为自己积攒财宝在地上……只要积攒财宝在天上……因为你的财宝在哪里，你的心也在哪里。」 → 你可能在地上贫穷，但如果你的眼光放在天国，你就是真正富足的人。

  • 启示录 2:9：「我知道你的患难和贫穷（你却是富足的）。」 → 地上的贫穷并不代表你在神眼中贫穷，在天国，你可能比地上的亿万富翁更富足！

✅ 现实生活应用：
不要只关注地上的贫富，而是关注天国的产业。有一天，你会在天国得着荣耀的冠冕，那是真正的财富！

3. 贫穷的基督徒应该如何走完这一生？

贫穷不是神的诅咒，而是神的拣选，让你更加经历祂的恩典。面对现实的困难，圣经给我们几个行动指南：

(1) 知足并感恩

  • 腓立比书 4:11-12：「我无论在什么景况，都可以知足……无论是处卑贱，还是处丰富，或是饱足，或是饥饿，我都得了秘诀。」 → 保罗经历过贫穷，但他仍然喜乐，因为他知道，神的恩典够用。

✅ 实际行动：

  • 每天感谢神供应你的一切，不管多或少。
  • 不要与富人比较，而是专注于神已经赐给你的祝福。

(2) 依靠神的供应

  • 马太福音 6:26：「你们看那天上的飞鸟，也不种，也不收，也不积蓄在仓里，你们的天父尚且养活它们。」 → 神关心飞鸟，更会供应你！

✅ 实际行动：

  • 无论生活有多难，祷告求神供应你的需要，祂必会看顾你。

(3) 不要惧怕未来

  • 希伯来书 13:5：「你们存心不可贪爱钱财，要以自己所有的为足；因为主曾说：‘我总不撇下你，也不丢弃你。’」 → 你可能贫穷，但神永远不会离开你，你永远不会是孤单的。

✅ 实际行动：

  • 无论面对什么挑战，都相信神与你同在。
  • 把未来交托给神，相信祂会带领你走完全程。

4. 结论

✔ 神拣选贫穷人，不是要他们受苦，而是要他们在信上富足，得着天国的产业。

  • 贫穷人更容易倚靠神，他们比富人更容易进天国。
  • 信心比金钱更宝贵，贫穷不能限制你在神国的财富。
  • 地上的贫富不重要，天国的产业才是永恒的财富。

✔ 作为贫穷的基督徒，你可以这样走完人生：

  1. 知足感恩，相信神已经赐给你足够的恩典。
  2. 依靠神的供应，不为生活忧虑，因神必看顾你。
  3. 盼望天国的赏赐，知道你的产业不在地上，而在天上。

「贫穷的人有福了！因为神的国是他们的。」（路加福音 6:20）

➡ 你在世上的贫穷不会永远持续，但你在基督里的富足却是永远的！
