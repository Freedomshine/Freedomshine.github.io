圣徒saint 与 生命的更新

Mar 08, 2025

“圣徒”（saint）这个概念在不同的基督教传统中有不同的理解，但核心含义是指那些归属于上帝、被神分别出来的人。以下是几种主要的解释方式：

1. 圣经中的“圣徒”

在圣经中，“圣徒”通常指所有真正信靠耶稣基督、因信称义的基督徒。例如：

  • 保罗书信中，他经常称写信对象为“圣徒”（如罗马书 1:7，哥林多前书 1:2），即所有信徒。
  • 圣徒的核心特征是被“分别为圣”（希腊文 hagios），意味着他们属于神，并过着敬虔的生活。

2. 天主教与东正教的“圣徒”

在天主教和东正教中，“圣徒”不仅指普通信徒，还特别指那些因信仰和善行而被教会正式承认、进入“圣品”的人。教会通常会有**封圣（Canonization）**的程序，确认某人具有卓越的信仰、行过神迹，或者殉道。

3. 更正教（新教）对圣徒的理解

新教通常不设立特定的“圣徒”阶层，而是回归圣经的用法，认为所有重生得救的基督徒都是圣徒。这意味着圣徒不只是少数“特别圣洁”的人，而是所有因信称义、在基督里被分别为圣的人。

4. 圣徒的实际意义

无论哪种理解，圣徒的核心意义在于：

  • 他们属于神，蒙神拣选。
  • 他们因耶稣基督的救赎被洁净、成圣。
  • 他们活出与世人不同的生命，追求圣洁和神的旨意。

如果是新教背景，可以简单理解：“每一个真正信靠耶稣基督、因信得救的人，都是圣徒。”

圣徒这个词感觉容易让人骄傲啊

确实，“圣徒”这个词如果误解，可能让人产生骄傲的心态，觉得自己比别人更圣洁、更优越。但如果按照圣经的真正含义理解，它反而应该让人更加谦卑。这里有几个原因：

1. “圣徒”不是靠自己得来的，而是神的恩典

  • 圣经强调，人成为圣徒不是因为自己的行为，而是因为神的拣选和恩典（以弗所书 2:8-9）。
  • 这意味着，我们被称为圣徒，不是因为我们配得，而是因为基督的救赎使我们成圣。

2. 圣徒意味着责任，而不是特权

  • 成为圣徒不是炫耀的资本，而是一个呼召，要活出圣洁的生命（彼得前书 1:15-16）。
  • 这是一种责任，要求谦卑顺服神，而不是骄傲自夸。

3. 圣徒的榜样是谦卑的

  • 圣经中的圣徒，真正属灵成熟的人，都是极其谦卑的。
  • 保罗称自己是“罪人中的罪魁”（提摩太前书 1:15），摩西被形容为“世上最谦和的人”（民数记 12:3）。
  • 他们之所以伟大，不是因为自己的能力，而是因为他们完全依靠神。

4. 圣徒是软弱但被神更新的人

  • 圣徒并不是指“完美无罪”的人，而是那些在基督里不断被神塑造、更新的人。
  • 这种身份应该让人感恩，而不是自豪，因为一切荣耀都归于神，而不是我们自己。

所以，正确理解圣徒的意义，不会让人骄傲，反而会让人更谦卑、感恩，并且努力活出圣洁的生命，以荣耀神。

我们能知道谁是圣徒在教会，因为只有主才能区分麦子和稗子

你说得很对！只有主才能完全区分谁是真正的圣徒，因为人的外表和行为并不总能反映内心的真实状态。耶稣在《马太福音》13:24-30 的稗子比喻里清楚说明，在教会里，真信徒（麦子）和假信徒（稗子）会一起生长，直到末日，主才会彻底分辨和审判。

1. 为什么我们不能完全分辨？

  • 人只能看外表，神才察看内心（撒母耳记上 16:7）。
  • 有些人外表敬虔，但可能没有真正得救（马太福音 7:21-23）。
  • 有些人看似软弱，甚至失败，但神仍然在他们生命中做工（如彼得一度跌倒，但后来成为教会的柱石）。

2. 但教会也不是完全没有分辨

虽然我们不能像神一样完全知道谁是真正的圣徒，但圣经给出了一些可见的记号：

  • 信仰告白：真圣徒承认耶稣基督为主，并信靠他的救恩（罗马书 10:9-10）。
  • 生命的果子：耶稣说：“凭着他们的果子，就可以认出他们来”（马太福音 7:16-20）。这包括：
    • 爱神、爱人（马可福音 12:30-31）
    • 追求圣洁（希伯来书 12:14）
    • 结出圣灵的果子（加拉太书 5:22-23）
  • 对真理的持守：真正的圣徒不会偏离基督的福音（约翰一书 2:19）。

3. 我们的态度：谦卑、忍耐、警醒

  • 不要论断别人得救与否，但可以根据圣经的教导鼓励彼此行在信仰中。
  • 耐心等候主的工作，因为神的计划有时候不是立即显明的（彼得后书 3:9）。
  • 自我省察：与其怀疑别人是否是圣徒，更重要的是问自己：“我是否在基督里？”（哥林多后书 13:5）。

4. 最终的审判属于神

在末日，神会完全显明谁是属他的（马太福音 25:31-46）。所以，我们应该用爱心建立教会，而不是忙着分辨谁是真正的圣徒，而是鼓励自己和他人真正活出圣徒的生命。

你的观点很符合圣经的教导：教会确实有看不见的“圣徒”，但最终，只有主能完全区分麦子和稗子。我们的责任是忠心传扬福音，活出圣洁，并彼此相爱，直到主再来。

人是会慢慢变化的，可能现在不是很好，但是将来会悔改归向主。

是的，人是会慢慢变化的，神的恩典能改变人的生命。现在看起来软弱、不信的人，将来也可能悔改归向主。圣经中有许多例子证明了这一点：

1. 圣经中的悔改例子

  • 彼得：曾三次否认主，但后来痛悔认罪，被主恢复，最终成为初代教会的领袖（约翰福音 21:15-19）。
  • 保罗：原本逼迫基督徒，但神在大马士革路上呼召他，他成为外邦人的使徒（使徒行传 9:1-19）。
  • 马太和撒该：原本是被人厌恶的税吏，但耶稣接纳他们，他们悔改跟随主（马太福音 9:9-13，路加福音 19:1-10）。
  • 浪子回头：耶稣的比喻表明，即使一个人远离神，神仍然等待他悔改归回（路加福音 15:11-32）。

2. 神有耐心，人可以慢慢改变

  • 神不愿一人沉沦，乃愿人人都悔改（彼得后书 3:9）。
  • 信徒生命的成长是一个过程，需要神的塑造（腓立比书 1:6）。
  • 有些人可能刚开始信主时软弱，但神会逐步更新他们（哥林多后书 3:18）。

3. 作为基督徒，我们的态度

  • 不要过早论断人：一个人现在不信主，不代表他一生都不会信主。
  • 要有耐心和爱心：我们要向周围的人活出基督的爱，而不是急于定论他们的属灵状况。
  • 要相信神的工作：许多信徒的生命是渐渐被神改变的，而不是一夜之间完全更新。

4. 祷告与信心

  • 为未信的人代祷，相信神会在他们生命中动工。
  • 鼓励那些信心软弱的人，因为神的恩典够他们用（哥林多后书 12:9）。
  • 信靠神的时间：神的计划比我们想象的更长远，我们只需忠心撒种，结果交给神（哥林多前书 3:6-7）。

所以，不要轻易论断一个人现在的状态，因为神的大能能改变任何人的生命，就像改变了彼得、保罗、撒该……我们只要忠心传扬福音，活出基督的样式，耐心等候神的工作。

所以我们最好不要根据这个人当前的属灵状态就妄下结论，也不应该因为一个人的属灵状态差就歧视他。

完全正确！我们不应该根据一个人当前的属灵状态妄下结论，也不应该因为他的属灵状态差就歧视他。这不仅是因为人会变化，更是因为圣经教导我们要用爱心、忍耐和盼望来看待别人，而不是论断或歧视。

1. 圣经的教导：不要论断，而要带着爱心对待人

a) 只有神知道人的内心和最终结局

  • 撒母耳记上 16:7：
    “耶和华不像人看人，人是看外貌，耶和华是看内心。”
  • 马太福音 7:1-2：
    “你们不要论断人，免得你们被论断。”
  • 罗马书 14:4：
    “你是谁，竟论断别人的仆人？他或站住，或跌倒，自有他的主人在。”
    → 这说明，每个人的属灵光景是神在掌管，我们不应该随意定论谁是“好”或“坏”。

b) 我们都是神恩典下的罪人

  • 罗马书 3:23：
    “因为世人都犯了罪，亏缺了神的荣耀。”
  • 以弗所书 2:8-9：
    “你们得救是本乎恩，也因着信，并不是出于自己，乃是神所赐的；也不是出于行为，免得有人自夸。”
  • 哥林多前书 1:27-29：
    “神却拣选了世上愚拙的，叫有智慧的羞愧；又拣选了世上软弱的，叫强壮的羞愧。”

→ 这意味着，我们今天能站立，不是因为我们比别人好，而是完全出于神的恩典。如果神恩待了我们，我们有什么资格去看不起别人呢？

2. 属灵生命是成长的过程

  • 许多信徒的信仰并不是一下子变得成熟，而是经历时间的塑造：

    • 彼得曾经三次不认主，但神改变了他（约翰福音 21:15-19）。
    • 约拿不愿意传福音给尼尼微人，但神耐心地管教他。
    • 许多信徒一开始很软弱，但在神的带领下成长。

  • 腓立比书 1:6：
    “我深信那在你们心里动了善工的，必成全这工，直到耶稣基督的日子。”
    → 这说明，属灵成长是一个持续的过程，我们应该耐心等待神在别人生命中的工作。

3. 如何正确对待属灵状态较差的人？

a) 以爱心帮助，而不是论断

  • 加拉太书 6:1：
    “弟兄们，若有人偶然被过犯所胜，你们属灵的人就当用温柔的心把他挽回；又当自己小心，恐怕也被引诱。” → 如果有人在信仰上软弱，我们要温柔地帮助他们，而不是批评、歧视或远离他们。

b) 耐心祷告，信靠神的作为

  • 提摩太前书 2:1：
    “我劝你，第一要为万人恳求、祷告、代求、祝谢。”
    → 有些人现在信仰软弱，不代表他们永远不会悔改归向主，我们要为他们祷告。

c) 以谦卑的心与人相处

  • 腓立比书 2:3：
    “凡事不可结党，不可贪图虚浮的荣耀；只要存心谦卑，各人看别人比自己强。”
    → 这提醒我们，不要因为自己属灵状态好一点，就觉得自己比别人强，而是要存谦卑的心。

4. 反思：我们自己也不是完全的

我们如果今天能站立，是因为神的恩典，而不是因为自己比别人更好：

  • 哥林多前书 10:12：
    “所以，自己以为站得稳的，须要谨慎，免得跌倒。”
  • 马太福音 18:21-22：
    “彼得进前来，对耶稣说：‘主啊，我弟兄得罪我，我当饶恕他几次呢？到七次可以吗？’耶稣说：‘我对你说，不是到七次，乃是到七十个七次。’”
    → 这提醒我们，神愿意饶恕和等待我们，我们也应该愿意等待别人悔改。

结论：用爱心代替论断，用耐心等候神的工作

  • 不要因人的属灵软弱就轻易定论，因为神的恩典能改变任何人。
  • 不要歧视信仰软弱的人，而是用爱心帮助他们，鼓励他们在信仰中成长。
  • 要相信神的计划，每个人属灵成长的节奏不同，我们要耐心等待。
  • 要谦卑，因为我们自己也是靠神的恩典才能站立，而不是靠自己的能力。

最终，神才是审判的主，我们的责任是去爱、去服侍、去等待，而不是去论断。
